Anything → Markdown

Synthesize clean Markdown from messy inputs: HTML, PDFs, YouTube pages, Wikipedia, and plaintext — with a tiny, powerful CLI and TypeScript API.

TypeScript • Node 18+

Smart HTML → MD

Sanitizes and converts with semantic headings and links preserved.

Reliable PDF Text

Extracts metadata and full text while respecting PDF structure.

Extensible

Add converters for new formats or websites through a simple interface.

Open Source
Contribute on GitHub

Quick examples

// CLI
npx anything-to-markdown https://en.wikipedia.org/wiki/TypeScript -o out.md

// Programmatic
import { MarkdownConverter } from 'anything-to-markdown';
const c = new MarkdownConverter();
const r = await c.convert('doc.pdf');
console.log(r.textContent);
